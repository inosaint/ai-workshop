# GitHub Account

GitHub is where developers store their code. Think of it as **Google Drive for code** - but with superpowers.

In this section, you'll create a free GitHub account that you'll use for the rest of the workshop.

---

## What is GitHub?

Before we dive in, let's understand what GitHub actually is:

| Concept | Everyday Analogy |
|---------|------------------|
| **GitHub** | Like Google Drive, but specifically designed for code projects |
| **Repository** (repo) | A folder that contains your project files |
| **Commit** | Saving a version of your work (like "Save As" with a note) |
| **Push** | Uploading your saved work to the cloud |
| **Pull** | Downloading the latest version from the cloud |

You don't need to memorize these now - they'll make more sense once we use them!

### Step 1: Go to GitHub

Open your browser and go to **[github.com](https://github.com)**

### Step 2: Click "Sign up"

Look for the **"Sign up"** button in the top-right corner and click it.

### Step 3: Enter Your Email

Type in your email address. This will be linked to your account, so use one you check regularly.

### Step 5: Choose a Username

This is your identity on GitHub! Some tips:
- Keep it professional (especially if you'll use this for work)
- Make it memorable
- It can be your real name, a nickname, or something creative

Examples: `johndoe`, `jane-codes`, `curious-builder`

GitHub will send a code to your email. Check your inbox (and spam folder!) for an email from GitHub.

Enter the code to verify your account.

## Configuring Your Profile (Optional)

While not required, adding a profile picture and bio helps others recognize you. You can skip this for now and come back later.

1. Click your profile icon (top-right corner)
2. Click **"Settings"**
3. Add a profile picture and bio if you'd like
