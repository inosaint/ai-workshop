# GitHub Account

GitHub is where developers store their code. Think of it as **Google Drive for code**.

In this section, you'll create a free GitHub account that you'll use for the rest of the workshop.

---

## A crash course of terms

Before we dive in, let's understand a few terms you would need to be familiar with.

| Concept               | Everyday Analogy                                                      |
| --------------------- | --------------------------------------------------------------------- |
| **GitHub**            | Like Google Drive(Cloud), but specifically designed for code projects |
| **Repository** (repo) | A folder that contains your project files                             |
| **Commit**            | Saving a version of your work (like "Save As" with a note)            |
| **Push**              | Uploading your saved work to the cloud                                |
| **Pull**              | Downloading the latest version from the cloud                         |

You don't need to memorize these now - they'll make more sense once we use them!

---

## Creating Your GitHub Account

### Step 1:
Head over to to **[github.com](https://github.com)** and create your own account. Choose a good username when creating your account 

### Step 2:
Download Github Desktop and login with your newly created credentials.

### Step 3:
Create a new repository(repo) using Github Desktop and Publish it.
Name it using the following format: [username].github.io 

### Step 4:
Head over to Github.com and Navigate to your newly created repository and click on Settings.

### Step 5:
Navigate to pages on the left nav and select, 'Deploy from a Branch'
