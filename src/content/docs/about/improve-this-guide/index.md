## How to Raise an Issue

1. **Check existing issues** to see if your concern has already been reported
2. **Click "New Issue"** on the [Issues page](https://github.com/inosaint/ai-workshop/issues/new)
3. **Choose a template** (or start from scratch)
4. **Provide details**:
   - Clear title describing the issue or suggestion
   - Which page or section you're referring to
   - What you expected vs. what you found
   - Screenshots if helpful
5. **Submit** and we'll review it as soon as possible!

## Types of Contributions We Welcome

- **Content improvements**: Better explanations, additional examples, or clearer instructions
- **Technical corrections**: Fixes for code errors, outdated commands, or broken links
- **New content**: Suggestions for additional modules or advanced topics
- **Accessibility**: Help us make the guide more accessible to all learners
- **Translations**: Interest in translating the guide to other languages

## Quick Fixes

Found a small typo or error? You can also:
- Use the **"Edit this page"** link at the bottom of any page
- Make the fix directly and submit a pull request

## Questions?

If you're not sure whether to create an issue or just have a general question, feel free to open an issue anyway. We're here to help!
