## Who Is This For?

This workshop is designed for **complete beginners** - no coding experience required. If you can:

- Use a web browser
- Type on a keyboard
- Follow step-by-step instructions

...then you have everything you need to succeed!

---

## Workshop Duration

**Approximately 3-3.5 hours** with breaks

| Part | Duration | What You'll Do |
|------|----------|----------------|
| Setup | ~45 min | Create accounts, install tools |
| Build | ~60 min | Create your project with Claude Code |
| Deploy | ~45 min | Go live on the internet |
| Level Up | ~45 min | Learn APIs and analytics |
